# Roadmap

## Phase 0: IPS baseline adoption
Create protected intent docs, governance docs, task/plan/context/validation artifacts, graph and local gates.
## Phase 1: Stabilize existing production integration
Keep offer, stock and order flows aligned with documented ownership boundaries.
## Phase 2: Marketplace capability improvements
Add Aukro capabilities only through feature docs, ADRs when needed and validated execution plans.
## Phase 3: Operational hardening
Improve observability, retries, contract tests and deployment evidence.
## Phase 4: Integration review
Review contracts with catalog, warehouse and orders before major changes.
