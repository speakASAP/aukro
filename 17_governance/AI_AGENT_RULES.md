# AI Agent Rules

## Required workflow

Read protected intent docs, task docs, execution plan, context package and validation criteria before implementation.

## Forbidden actions

Do not modify protected intent docs after baseline adoption, change runtime behavior without an execution plan, expose secrets/raw production data, invent goals, remove traceability or alter deployment manifests outside scope.

## Final report

Report changed files, documents created, sections filled, remaining markers, validation evidence and deviations.
